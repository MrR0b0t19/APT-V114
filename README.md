# APT-Vi114
villa HACKGDL


Documentación completa de un ataque Red Team a infraestructura hospitalaria:

desde la explotación de un archivo DICOM malicioso en un mastógrafo, pasando por reverse shell en estación médica Windows, escalación a SYSTEM, para después realizar un volcado de credenciales LSASS, hasta el compromiso total de Active Directory mediante una cadena de 7 fases de abuso de ACLs (WriteDACL → AddMember → GenericWrite → DCSync). Cada fase documentada en formato profesional de 4 partes: explicación técnica, lógica del ataque, remediación y ejercicio práctico. Resultado final: Control total del dominio hospitalario desde un archivo de imagen médica y que la audiencia sea capaz de detectar, proteger y replicar toda la parte ofensiva ya que se compartirá todo el código desde.
